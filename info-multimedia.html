Guía de Contenido Multimedia en HTML5

Aprende a integrar imágenes semánticas, audio, video y elementos externos embebidos de manera profesional.

1. Imágenes Semánticas (<figure> y <figcaption>)

En HTML5, cuando una imagen está acompañada de un texto explicativo o ilustrativo, se agrupa dentro de la etiqueta <figure> para darle un contexto semántico que los buscadores puedan indexar correctamente.

Código Fuente

<figure>
    <img src="paisaje.jpg" alt="Montañas nevadas al amanecer" width="100%">
    <figcaption>Fig.1 - Vista panorámica de los Alpes suizos en invierno.</figcaption>
</figure>
Muestra de Texto Explicativa (Resultado Esperado)
[ Espacio de renderizado de la imagen: paisaje.jpg ]

Fig.1 - Vista panorámica de los Alpes suizos en invierno. <figcaption>

Beneficio: Vincula de forma directa el texto descriptivo con el archivo gráfico, mejorando la accesibilidad y el SEO de la página en comparación a usar un simple párrafo suelto debajo de la imagen.

2. Reproductor de Audio Nativo (<audio>)

Permite reproducir música o podcasts sin necesidad de plugins externos (como el antiguo Flash). Soporta múltiples formatos y utiliza la etiqueta interna <source> como alternativa en caso de incompatibilidad de formatos.

Código Fuente

<audio controls>
    <source src="podcast.mp3" type="audio/mpeg">
    <source src="podcast.ogg" type="audio/ogg">
    Tu navegador no soporta la reproducción de audio nativo.
</audio>
Muestra de Texto Explicativa (Resultado Esperado)
▶
01:24 / 05:00 🔊
  • controls: Atributo clave. Si no se incluye, el reproductor será completamente invisible para el usuario.
  • Texto de respaldo: La frase "Tu navegador no soporta..." solo se visualizará si el usuario utiliza un navegador extremadamente antiguo que no entienda la etiqueta.

3. Reproductor de Video Nativo (<video>)

Similar al elemento de audio, pero añade capacidades visuales. Permite controlar dimensiones, imágenes de portada (posters) y comportamientos de reproducción.

Código Fuente

<video src="tutorial.mp4" controls poster="miniatura.jpg" width="480">
    Su navegador no puede reproducir este video.
</video>
Muestra de Texto Explicativa (Resultado Esperado)
[ Aquí se muestra el archivo miniatura.jpg como portada (Poster) ]
▶
⏸
00:45 / 12:30 HD ⚙ 🔲

Atributos Adicionales Comunes:
• autoplay: El video inicia solo (los navegadores modernos exigen que esté silenciado para permitirlo).
• loop: Hace que el video se reinicie infinitamente al terminar.
• muted: Desactiva el sonido por defecto.

4. Contenido Embebido Externo (<iframe>)

Un iFrame (Inline Frame) funciona como una "ventana" que incrusta un documento web completamente diferente o servicios de terceros (como mapas de Google Maps o videos de YouTube) dentro de tu propia página.

Código Fuente

<iframe src="https://www.wikipedia.org" width="100%" height="300" title="Enciclopedia Wikipedia"></iframe>
Muestra de Texto Explicativa (Resultado Esperado)
🔒 Ventana incrustada: https://www.wikipedia.org

Wikipedia

[ Todo el sitio web de Wikipedia se cargaría y operaría de manera interactiva exactamente en este espacio interior sin salirse de tu diseño corporativo ]

Nota Importante de Accesibilidad: El atributo title="..." es sumamente importante en los iFrames. Informa a los lectores de pantalla qué contenido se está integrando desde fuera del dominio principal.
