Guía de Elementos HTML y Muestras Prácticas

Aprende la estructura web con explicaciones semánticas y ejemplos de código interactivos enriquecidos con Prism.js.

1. Estructura Básica de un Documento

Todo archivo HTML requiere una declaración de tipo y etiquetas raíz que envuelven el encabezado (metadatos) y el cuerpo (contenido visual).

Código Fuente

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Primera Página Web</title>
</head>
<body>
    <!-- El contenido de la página se añade aquí -->
</body>
</html>
Explicación del Comportamiento
  • <!DOCTYPE html>: Indica al navegador que use el estándar HTML5.
  • <html lang="es">: Define el idioma, permitiendo a los motores de búsqueda e interfaces de accesibilidad traducir o leer correctamente.
  • <head>: Aloja configuraciones invisibles (como la codificación de caracteres UTF-8 para aceptar acentos y la "ñ").
  • <body>: Contenedor maestro de todo lo que el usuario final verá en pantalla.

2. Encabezados Jerárquicos (<h1> a <h6>)

Establecen la importancia de los títulos de tu contenido. Tienen un impacto crítico en el SEO (posicionamiento en buscadores).

Código Fuente

<h1>Diseño Web Profesional</h1>
<h2>Introducción al Desarrollo Frontend</h2>
<h3>Conceptos Clave de HTML</h3>
Muestra de Texto Explicativa (Resultado Esperado)

Diseño Web Profesional

Este es el encabezado principal (<h1>). Solo debe haber uno por página web.

Introducción al Desarrollo Frontend

Subtítulo de segundo nivel (<h2>) usado para separar las secciones principales.

Conceptos Clave de HTML

Sub-sección de tercer nivel (<h3>) para detallar subtemas específicos.

3. Párrafos y Formato de Texto (<p>, <strong>, <em>)

Los textos genéricos usan bloques separados automáticamente. Dentro de ellos se puede añadir peso semántico para destacar palabras.

Código Fuente

<p>Para dominar la web, es <strong>fundamental comprender HTML</strong>.</p>
<p>Recuerda la regla: <em>"El código limpio es oro"</em>.</p>
Muestra de Texto Explicativa (Resultado Esperado)

Para dominar la web, es fundamental comprender HTML Texto en negrita vía <strong>.

Recuerda la regla: "El código limpio es oro" Texto en cursiva vía <em>.

4. Enlaces e Hipervínculos (<a>)

Conectan tu documento con páginas externas o anclas internas empleando el atributo indispensable href.

Código Fuente

<p>Visita nuestro <a href="https://www.ejemplo.com" target="_blank">sitio web oficial</a>.</p>
Muestra de Texto Explicativa (Resultado Esperado)

Visita nuestro sitio web oficial.

Comportamiento práctico: El texto se vuelve interactivo, cambia a color azul por defecto y se subraya. El parámetro target="_blank" fuerza al sistema a abrir la dirección en una pestaña nueva.

5. Imágenes (<img>)

Inserta recursos gráficos en la página. No cuenta con etiqueta de cierre (es un elemento vacío).

Código Fuente

<img src="espacio-trabajo.jpg" alt="Mesa de diseño con laptop y café" width="400">
Análisis de Atributos Críticos
  • src: Especifica el origen o ruta física/URL de donde se descargará el archivo visual.
  • alt (Texto Alternativo): Si la imagen no carga o el usuario es invidente y usa lectores de voz, el sistema leerá textualmente: "Mesa de diseño con laptop y café". ¡Es obligatorio para la accesibilidad!

6. Listas Ordenadas y Desordenadas

Permiten enumerar agrupaciones de datos de manera ordenada (numérica) o desordenada (con viñetas redondeadas).

Código Fuente

<!-- Lista Desordenada -->
<ul>
    <li>HTML5</li>
    <li>CSS3</li>
</ul>

<!-- Lista Ordenada -->
<ol>
    <li>Fase de Maquetación</li>
    <li>Fase de Estilos</li>
</ol>
Muestra de Texto Explicativa (Resultado Esperado)

Salida del contenedor <ul>:

  • HTML5 (Viñeta asignada automáticamente)
  • CSS3

Salida del contenedor <ol>:

  1. Fase de Maquetación (Número secuencial generado de manera automática)
  2. Fase de Estilos

7. Tablas de Datos (<table>)

Estructura información compleja en una matriz organizada en filas (<tr>), títulos de columna (<th>) y celdas de datos comunes (<td>).

Código Fuente

<table>
    <tr>
        <th>Módulo</th>
        <th>Horas</th>
    </tr>
    <tr>
        <td>HTML Básico</td>
        <td>10h</td>
    </tr>
</table>
Muestra de Texto Explicativa (Resultado Esperado)
Módulo Horas
HTML Básico 10h

Las celdas de tipo <th> se renderizan en negrita y centradas de manera nativa para resaltar que actúan como cabeceras.

8. Formularios e Interacción (<form>, <input>)

Son los elementos encargados de capturar las acciones del usuario y enviar texto o datos clave al servidor backend.

Código Fuente

<form>
    <label for="user">Usuario:</label>
    <input type="text" id="user" placeholder="Ej: Pepe">
    
    <button type="submit">Enviar</button>
</form>
Muestra de Texto Explicativa (Resultado Esperado)

El atributo placeholder inserta un texto guía semitransparente que se borra de manera automática en cuanto el usuario presiona una tecla dentro del campo.
